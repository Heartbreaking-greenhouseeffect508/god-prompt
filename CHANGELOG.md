# Changelog

All notable changes to GodPrompt will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-03-21

### Added
- **SKILL.md** — 248-line core with Three Iron Laws, 6-phase universal protocol, auto-classification for 9 task types
- **protocols.md** — deep execution guides: BUILD (TDD), DEBUG (root cause + 4-layer defense), REFACTOR, CONTENT (SEO + affiliate), DESIGN (anti-AI-slop), SHIP, ANALYZE, AUTOMATE, PLAN
- **anti-patterns.md** — red flags, rationalization prevention tables, partner signal recognition, test anti-patterns, recovery playbook
- **quality-gates.md** — verification checklists per task type, universal gate (5 checks every task), delivery report template
- **GodPrompt.md** —  the 1,100+ line single-file payload for quick deployment
- MIT license, contributing guide, changelog

### Design Decisions
- Progressive disclosure: 248-line core always in context, reference files loaded on-demand
- Auto-classification replaces manual skill selection
- Unified protocol phases (classify → understand → plan → execute → verify → deliver) replace per-skill workflows
- Distilled from 30+ individual skills (safe-implementation, systematic-debugging, verification-before-completion, test-driven-development, defense-in-depth, brainstorming, frontend-design, root-cause-tracing, and more)
